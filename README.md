## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/LTPPPP/FBotder.git

2. Install library for python
   ```sh
   pip install -r requirements.txt
3. Change directory
   ```sh
   cd python
4. Start the project
   ```sh
   python geminiBot.py
5. Run url for public
   ```sh
